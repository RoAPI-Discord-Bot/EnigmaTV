# EnigmaTV

Unified Android streaming app for movies and TV — one home screen, sidebar navigation, favorites, playlists, and protected WebView playback.

## Features

- **Unified home** — trending & popular movies and TV together; one search for both
- **Sidebar** — Home, Favorites, Continue Watching, My Lists (playlists)
- **Favorites** — heart on any title; persisted locally
- **Playlists** — create named lists and add titles from favorites
- **Loading UI** — EnigmaTV logo with purple/pink spinning ring (app launch & stream connect)
- **Redirect protection** — blocks hijack navigations and popups in the player

## Stream sources

**Movies:** VidLink, Vidsrc.to, Vsembed, Vidsrc.cc, Vidsrc.me, Embed.su  

**TV:** VidLink, Vidsrc.to, Vidsrc.cc, Vidsrc.me, Embed.su, MultiEmbed

## Build

Open in Android Studio or use GitHub Actions (`.github/workflows/android.yml`) for a debug APK artifact.

## Usage

1. Tap **menu** (☰) to open the sidebar.
2. Browse everything on **Home** — no switching between separate movie/TV apps.
3. Tap **♥** on a poster to favorite.
4. **My Lists** → **+** to create a playlist; add items from **Favorites** via the playlist icon.
